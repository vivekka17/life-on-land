# 🌱 Life on Land – SDG 15 Front-End Web Platform

A responsive front-end web platform developed to support **United Nations Sustainable Development Goal 15: Life on Land**. The project focuses on environmental conservation, sustainable land management, biodiversity awareness, and community engagement through modern and accessible web technologies.

---

## 📌 Project Overview

The platform was designed to raise awareness about land conservation and environmental sustainability while providing users with educational resources and interactive volunteer opportunities. The website combines responsive layouts, animated UI components, accessibility-focused design, and engaging user experiences using front-end technologies.
